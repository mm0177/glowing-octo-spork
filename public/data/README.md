# Data Files

This directory contains runtime data served statically by Next.js.

## Files (generated — not committed to git)

- `personas.compact.india.json` — sampled Indian personas (generated by `scripts/prepare_personas_india.py`)
- `personas.compact.india.meta.json` — metadata about the sample
- `india-states.geojson` — India state boundaries (downloaded by `scripts/download_india_geojson.py`)

## Regenerating

```bash
# 1. Download India states map
python scripts/download_india_geojson.py

# 2. Prepare personas (requires HuggingFace access or local parquet files)
python scripts/prepare_personas_india.py --sample-size 5000 --seed 42
```
